# TaskFlow

Клиент-серверное веб-приложение (Single Page Application) для визуального планирования задач по методологии Канбан.

## Технологический стек

- **Backend:** Node.js, Express.js, fs (локальное файловое JSON-хранилище).
- **Frontend:** HTML5, Tailwind CSS, Vanilla JavaScript (ES6+), нативный HTML5 Drag-and-Drop API.

## Функционал

- Полный цикл CRUD-операций для задач (создание, чтение, редактирование, удаление).
- Изменение статуса задачи перетаскиванием карточки (Drag-and-Drop).
- Полнотекстовый поиск по названию и описанию в реальном времени.
- Фильтрация задач по уровням приоритета (Low, Medium, High).
- Динамический пересчет количества задач в колонках.
- Адаптивная верстка (мобильные и десктопные устройства).

## Установка и запуск

### 1. Запуск бэкенда
Перейдите в директорию бэкенда, установите зависимости и запустите сервер:

```
cd backend
npm install
npm start
```

Сервер будет запущен по адресу: `http://localhost:5000`

### 2. Запуск фронтенда
Откройте файл `frontend/index.html` напрямую в любом современном браузере. Для корректной работы клиентской части сервер бэкенда должен быть предварительно запущен.
